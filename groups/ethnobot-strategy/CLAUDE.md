## 🏗️ **Ethnobot Technical Context**

### **Current Stack**
- **Frontend**: Next.js 15 + TypeScript + Tailwind CSS
- **Backend**: Supabase (PostgreSQL + Auth + Edge Functions)
- **AI**: Claude API for interview analysis
- **Deployment**: Vercel (now on Australia region)
- **Monitoring**: Sentry + Axiom + PostHog

### **Core Architecture**
- **Two-Pool Data Model**: Discover (public profiles) + Org Studies (private research)
- **AI Interviewer**: Helix conducts natural conversations
- **Analysis Engine**: Claude extracts insights (AAA levels, trust, segments)
- **Multi-tenant**: Each organization has isolated data

### **Key Files I Work With**
- `/src/app/api/*` - API endpoints and business logic
- `/src/components/*` - React UI components
- `/src/lib/*` - Shared utilities and services
- `/supabase/migrations/*` - Database schema changes
- `/src/app/*/page.tsx` - Next.js pages
